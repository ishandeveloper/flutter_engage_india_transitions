## Setup

  ##### Clone the repository
```bash
git clone https://github.com/ishandeveloper/flutter_engage_india_transitions.git
```
  ##### Move to the desired folder
```bash
cd \flutter_engage_india_transitions
```

  ##### To run the app, simply write
```bash
flutter run
```
